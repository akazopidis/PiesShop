****Fully functional and responsive website in Jakarta EE 10 showcasing the MVC Design Pattern.****

**Technologies used:**

**Front-End: HTML, CSS, Sass, JavaScript**

**Back-End: Jakarta EE 10**

**Web server: Apache Tomcat**

**Database: MySQL**

**Deployment: GCP**

**Domain Provider: GoDaddy**
